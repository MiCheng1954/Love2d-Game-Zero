# Zero 游戏功能测试用例

> 本目录为**游戏端到端功能测试用例**，通过 `/autotest` 自动化执行。
> 与 `tests/` 下的单元测试（Lua/lupa）互补，覆盖实际游戏运行行为。

## 目录结构

```
tests/gameplay/
├── README.md               — 本文件，索引与规范
├── TC-01-menu.md           — 主菜单 & 角色选择
├── TC-02-combat.md         — 战斗核心（移动/敌人/子弹）
├── TC-03-weapons.md        — 武器系统（13把武器 + 射击效果）
├── TC-04-skills.md         — 技能系统（主动/被动技能）
├── TC-05-bag.md            — 背包系统（放置/移动/融合）
├── TC-06-synergy.md        — Tag 羁绊系统
├── TC-07-upgrade.md        — 升级奖励选择
├── TC-08-progression.md    — 局外成长界面（通用属性 + 机制树 + 技能树）
├── TC-09-gameover.md       — 游戏结算流程
└── TC-10-console.md        — 开发者控制台指令
```

## 用例状态说明

| 标记 | 含义 |
|------|------|
| ✅ | 通过 |
| ❌ | 失败（需修复） |
| ⚠️ | 异常（结果不确定，需人工确认） |
| ⏭ | 跳过（前置条件未满足） |

## 执行方式

由 `/autotest` skill 按 TC-01 → TC-10 顺序自动执行，每轮结束后修复 Bug 再重跑，直到全部通过。
