# Phase 13 开发计划 — 局外系统

## 目标
实现局外持久成长体系：角色/武器解锁、局外货币积累、永久属性加成、成就系统。

> ⚠️ 本阶段细节待与冰冰进一步讨论后再细化，当前为初步规划。

---

## 一、主要模块

### 1.1 局外货币与永久加成
- 局外货币（与局内灵魂区分，暂定名"魂晶"）
- 每局结算时根据表现获得魂晶
- 魂晶用于在局外商店购买永久加成（攻击/速度/HP等基础属性+%）
- 永久加成数据持久化存储至 `data/progression.json`

### 1.2 角色解锁
- 初始解锁1个角色，其余角色通过局外货币或特殊条件解锁
- 角色选择界面（主菜单入口）
- 每个角色有独特的：基础属性、专属技能、专属升级选项

### 1.3 武器解锁
- 部分稀有武器需要通过局外条件解锁才能在局内出现
- 解锁后加入武器池，升级选项中可随机出现

### 1.4 成就系统
- 成就配置表（config/achievements.lua）
- 成就条件：击杀数/存活时长/融合特定武器/激活特定羁绊等
- 成就完成解锁特殊内容（头像框/称号/稀有武器等）
- 成就数据持久化至 `data/achievements.json`

### 1.5 局外主界面
- 主菜单扩展：新增"传承"/"成就"/"图鉴"入口
- 图鉴：已解锁的武器/技能/敌人/Boss 说明
