# Phase 7 开发计划 — 武器融合与羁绊

## 目标
实现武器相邻增益、武器融合配方、武器羁绊系统，并升级索敌逻辑。

---

## 一、主要模块

### 1.1 相邻增益（Adjacency Bonus）
- 背包中相邻放置的武器互相提供加成效果
- 增益类型：攻击力+%、射速+%、射程+% 等
- 每种武器定义自己能给相邻武器提供什么增益，以及能从相邻武器获取什么增益
- Phase 6 已在 `Bag` 中预留接口，本阶段实装
- 计算时机：每次背包发生变化（放置/移除/移动）时重新计算并缓存

### 1.2 武器融合（Weapon Fusion）
- N 把武器可合成新武器（如 pistol + smg = 双持手枪）
- 新建 `config/fusion.lua`：融合配方表，`{ingredients=[...], result=weaperID}`
- 融合触发时机：背包界面检测到满足配方条件时提示玩家
- 融合后原材料武器销毁，新武器放入背包（需有空位或自动替换原位）
- 融合武器需扩充 `config/weapons.lua`，新增融合产物的配置

### 1.3 武器羁绊系统（Weapon Synergy）
- 背包中同时拥有特定武器组合时，激活羁绊效果
- 新建 `config/synergies.lua`：羁绊配置表
- 羁绊效果为全局 buff（不局限于相邻格）
- HUD/背包界面显示当前激活的羁绊

### 1.4 索敌逻辑升级
- 替换 `game.lua` 中的 `_findNearestEnemyInRange()` 占位实现
- 支持锁定策略：最近/最低血量/最高威胁
- 为 Phase 8 技能系统的索敌需求预留接口
