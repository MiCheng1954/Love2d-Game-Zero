# Phase 3 开发计划 — 敌人、战斗与自动攻击

## 目标
接入敌人系统、投射物、生成器、碰撞检测、自动攻击，实现完整的基础战斗循环。

## 二、各模块设计

### 2.1 config/enemies.lua
配置驱动，三种敌人：
- `basic`：平衡型，标准速度/HP
- `fast`：高速低HP
- `tank`：低速高HP高伤

### 2.2 enemy.lua
- 继承 Entity
- AI：每帧向玩家方向移动
- 接触伤害：有冷却时间，防止每帧连续伤害
- HP 条绘制（头顶）
- 死亡时生成掉落物

### 2.3 projectile.lua
- 直线飞行，超出最大距离自动销毁
- 命中敌人后销毁并触发伤害
- 支持暴击判定
- 代码绘制（含发光效果）

### 2.4 spawner.lua
- 按时间动态调整难度（生成间隔/批次数）
- 玩家周围随机方向生成，距离固定
- 实现节奏曲线：慢→快→非常快→慢

### 2.5 collision.lua
- 圆形碰撞检测
- 子弹 vs 敌人
- 敌人 vs 玩家（接触伤害）
- 死亡清理，返回掉落物数据

### 2.6 自动攻击（game.lua）
- 计时器驱动，每隔固定间隔锁定最近敌人
- 生成投射物飞向目标
- 调试面板（右上角）：FPS/等级/HP/攻击/敌人数

---

## 三、验证步骤
1. 敌人从四周生成，向玩家追击
2. 玩家自动锁定最近敌人，发射投射物
3. 子弹命中敌人，敌人扣血，HP条更新
4. 敌人接触玩家，玩家扣血
5. 玩家 HP 归零，跳转结算界面
6. 难度随时间动态提升
